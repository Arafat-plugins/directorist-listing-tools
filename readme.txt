Advanced bulk listing management tools for Directorist plugin.

== Description ==

Directorist Listing Tools provides powerful bulk management features for Directorist listings, making it easy for administrators to manage large numbers of listings efficiently.

== Features ==

* **Bulk Delete Listings**: Delete multiple listings at once by entering comma-separated listing IDs
* **Pending Listings Manager**: View and manage all pending listings with bulk publish or delete actions
* **Listing Type Manager**: Set or change listing types for multiple listings simultaneously

== Installation ==

1. Upload the `directorist-listing-tools` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Directorist → Listing Tools in the admin menu

== Requirements ==

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Directorist plugin (must be installed and activated)

== Frequently Asked Questions ==

= Do I need Directorist installed? =

Yes, this plugin requires Directorist to be installed and activated. The plugin will display a notice if Directorist is not active.

= What permissions are required? =

Only users with `manage_options` capability (typically administrators) can access these tools.

= Can I undo bulk deletions? =

No, bulk deletions are permanent. Please be careful when using the bulk delete feature.
